# Image Processing

批量图片处理工具，可在浏览器中直接选择图片、处理并下载，也保留本地 `input` / `output` 文件夹批处理方式。

在线地址：https://image-auto-processor.vercel.app

## 翻译工具

- 输入任意语言文本。
- 一键翻译成英语、德语、西班牙语、法语、意大利语、葡萄牙语、俄语、日语、阿拉伯语、韩语、土耳其语、泰语、越南语、荷兰语、印度尼西亚语、希伯来语、印地语、中文。
- 支持自动识别输入语言。

## 在线网页功能

- 选择整个文件夹里的 `.jpg`、`.jpeg`、`.png` 图片，可包含子文件夹。
- 支持 `1000x1000 白色画布居中`、`宽度 1920，高度等比缩放`、`宽度 750，高度等比缩放` 三种尺寸模式。
- 图片不拉伸、不变形。
- 图片主体居中。
- 输出 JPG，质量 95。
- 可开启去背功能，适合纯色或接近纯色背景；开启后输出透明 PNG。
- 输出文件名保持原文件名；普通模式扩展名为 `.jpg`，去背模式扩展名为 `.png`。
- 可选择另一个输出文件夹保存，处理后会保留原来的子文件夹结构。
- 图片在浏览器本地处理，不上传到服务器。

## 本地图形界面运行方法

直接双击：

```text
启动图形界面.bat
```

浏览器会自动打开图形界面。

## 本地命令行运行方法

1. 安装依赖：

```powershell
npm.cmd install
```

2. 把图片放到 `input` 文件夹。

3. 运行处理：

```powershell
npm.cmd run process
```

处理后的图片会出现在 `output` 文件夹。
